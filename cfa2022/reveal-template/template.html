    Un bloc important

      Plusieurs plugins sont activés:

    • La numérotation des slides en bas à gauche permet, en cliquant dessus, d'accéder rapidement au plan détaillé de la présentation
    • Le menu simple en haut permet d'avoir des liens vers des grandes parties de la présentation
    • Le plugin zoom (activé en faisant CTRL + clic sous Linux ou Alt+Clic sous Mac)
    • Le plugin de coloration syntaxique de code
    • Le plugin de recherche dans les slides (activé en faisant CTRL+Shift+F )

    Slides verticaux

    Il suffit d'imbriquer des sections dans des sections.

    Utiliser la touche Espace pour naviguer entre les slides


    Les diapositives imbriquées sont utiles pour ajouter des détails supplémentaires sous une diapositive horizontale de haut niveau.

    Appuyer sur la touche ECHAP pour avoir un overview sur les slides et leur organisation spatiale

    Slides cachés

    Cette diapositive est visible dans la source, mais cachée lorsque la présentation est affichée. Vous pouvez afficher toutes les diapositives cachées en mettant l'option de configuration `showHiddenSlides` à `true`.

    Global State

    Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

    State Events

    Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 

    
Reveal.on( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );

    Take a Moment

    Press B or . on your keyboard to pause the presentation. This is helpful when you're on stage and want to take distracting slides off the screen.
